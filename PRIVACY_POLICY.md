## 4. Data Security

We apply industry-standard security measures to protect your information:

### 4.1 Technical Security Measures

- **Transmission Encryption:** All data is transmitted encrypted via HTTPS/TLS
- **Storage Encryption:** Passwords are hashed and stored using the bcrypt algorithm
- **Token Security:** Authentication tokens are securely stored in iOS Keychain
- **Database Security:** Database access is through SSL encrypted connections

### 4.2 Security Limitations

No method of Internet transmission or electronic storage is 100% secure. While we use commercially reasonable means to protect your information, we cannot guarantee absolute security.
